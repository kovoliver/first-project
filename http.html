Mi a HTTP?

A HTTP (HyperText Markup Language) egy hálózati kommunikációs protokoll, amely egy úgynevezett user agent és egy webszerver közötti kommunikácót ír le.

HTTP kérés

A HTTP kérés első sora metódus erőforrás verzió alakú.

  • metódus: GET, POST, PUT, DELETE, HEAD, PATCH
  • erőforrás: az az erőforrás, amihez hozzá akarunk férni
  • verzió: a HTTP verziója (1.1 v 2.0)

Üzenettest

Az üzenetben fogadott/küldött adatok.

Fejlécállomány

A kérés vagy válasz beállításai és információk a küldőről.

HTTP metódusok

  • GET -> erőforrások lekérdezése
  • POST -> erőforrás létrehozása
  • PUT -> erőforrás teljes körű felülírása
  • PATCH -> részleges felülírás
  • DELETE -> törlés
  • HEAD -> ugyanaz, mint a GET, de üzenettest nélkül

HTTP válasz

A HTTP válasz első sora verzió státuszkód szöveges indoklás alakú.

  • verzió: HTTP verziója
  • státuszkód: információ a kérés állapotáról
  • szöveges indoklás: státuszkód emberi nyelven

Státuszkódok

  • 1xx: informatív
  • 2xx: OK
  • 3xx: átirányítás
  • 4xx: kliens hiba
  • 5xx: szerver hiba

AJAX

Az AJAX (Asynchronous JavaScript and XML) aszinkron HTTP kérések JavaScript-ben. Ez még hozzáadtam.
